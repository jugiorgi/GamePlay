## Sobre

É uma aplicação desenvolvida em React Native

##### Preparando o ambiente

- [x] Node + NPM
- [x] Yarn
- [x] Expo

##### Instalando as dependências

```sh
yarn install
```

##### Rodando a aplicação local (Android)

```sh
expo start
```
